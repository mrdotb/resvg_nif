# Resvg (Rust NIFs for elixir)

Native Implemented Function (NIF) bindings for the resvg library.

## Installation

The package can be installed
by adding `resvg_nif` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:resvg, "~> 0.1.0"}
  ]
end
```
